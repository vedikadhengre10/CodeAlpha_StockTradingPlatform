# CodeAlpha_StockTradingPlatform
A full-stack Stock Trading Platform developed for the CodeAlpha Internship Program. Features portfolio management, stock tracking, transaction history, price alerts, and an interactive dashboard built using Java, HTML, CSS, JavaScript, Node.js, and database integration.


# 📈 CodeAlpha Stock Trading Platform

A modern Stock Trading Platform developed as part of the **CodeAlpha Internship Program**.

---

## 🚀 Project Overview

The Stock Trading Platform is designed to provide users with a realistic trading experience while demonstrating core software development concepts.

---

## ✨ Features

### 📊 Trading Operations
- Buy stocks
- Sell stocks
- View available stocks

### 💼 Portfolio Management
- Track owned stocks
- View portfolio value

### 📜 Transaction History
- Record all buy/sell activities

### 🎨 User Interface
- Interactive dashboard
- Responsive design
